    Tugas 1 Penambangan Data¶

    1. Mean (Nilai Rata-Rata)

    Mean adalah teknik penjelasan kelompok yang didasarkan atas nilai rata-rata dari kelompok tersebut. Rata-Rata (mean) ini didapat dengan menjumlahkan data seluruh individu dalam kelompok itu, kemudian dibagi dengan jumlah individu yang ada pada kelompok tersebut

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["P"].describe()
df["Q"].describe()
df["R"].describe()
df["S"].describe()
print("reta-rata P :",df['P'].mean())
print("reta-rata Q :",df['Q'].mean())
print("reta-rata R :",df['R'].mean())
print("reta-rata S :",df['S'].mean())

    Output :

    reta-rata P : 4.396
reta-rata Q : 4.792
reta-rata R : 5.662
reta-rata S : 5.602

    2. Median (Nilai Tengah)

    median adalah salah satu teknik penjelasan kelompok yang didasarkan atas nilai tengah dari kelompok data yang telah disusun urutannya dari yang terkecil sampai yang terbesar, atau sebaliknya dari yang terbesar sampai yang terkecil.

    Contoh Median :

    Hasil observasi umur pegawai di kantor X adalah :

    20, 45, 60, 56, 45, 45, 20, 19, 57, 45, 45, 51, 35

    Untuk dapat mencari mediannya maka data umur diatas harus disusun terlebih dahulu urutannya. Setelah disusun, menjadi sebagai berikut :

    19, 20, 20, 35, 45, 45, 45, 45, 45, 51, 56, 57, 60

    Nilai tengah data diatas berada pada urutan ke 7 yaitu 45. Jadi mediannya adalah 45.

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["S"].describe()
a=("Q2 S",df['S'].median())
print (a)

    Output :

    ('Q2 S', 6.0)

    3. Modus (Nilai Yang Paling Banyak Muncul)

    modus adalah teknik penjelasan kelompok yang didasarkan atas nilai yang sedang populer (yang sedang menjadi mode) atau nilai yang sering muncul dalam kelompok tersebut.

    Contoh Modus Pada Data Kualitatif :

    Tahun 1970 di Yogyakarta, banyak mahasiswa yang naik sepeda. Sehingga dapat menjelaskan dengan modus, bahwa kelompok mahasiswa di Yogyakarta masih banyak yang naik sepeda.

    Contoh Modus Pada Data Kuantitatif :

    Umur pegawai kantor Y adalah :

    20, 45, 60, 56, 45, 45, 20, 19, 57, 45, 45, 51, 35

    Dari data diatas, dapat dilihat bahwa yang paling banyak muncul adalah umur 45. Munculnya sebanyak 5 kali, jadi dapat dijelaskan bahwa kelompok pegawai kantor Y sebagian besar berumur 45 tahun.

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["P"].describe()
df["Q"].describe()
df["R"].describe()
df["S"].describe()
print("modus P :",df['P'].mode())
print("modus Q :",df['Q'].mode())
print("modus R :",df['R'].mode())
print("modus S :",df['S'].mode())

    Output :

    modus P : 0    1
1    2
2    3
3    4
dtype: int64
modus Q : 0    2
1    3
2    4
3    5
4    6
dtype: int64
modus R : 0    8
dtype: int64
modus S : 0    7
dtype: int64

    4. Standar Deviasi

    Standar deviasi adalah nilai statistik yang digunakan untuk menentukan bagaimana sebaran data dalam sampel, dan seberapa dekat titik data individu ke mean – atau rata-rata – nilai sampel.

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["P"].describe()
df["Q"].describe()
df["R"].describe()
df["S"].describe()
print("standar deviasi P :",df['P'].std())
print("standar deviasi Q :",df['Q'].std())
print("standar deviasi R :",df['R'].std())
print("standar deviasi S :",df['S'].std())

    Output :

    standar deviasi P : 2.400564395894781
standar deviasi Q : 2.2827961618666848
standar deviasi R : 2.086255238901436
standar deviasi S : 2.382214493235982

    5. Varian

    varian adalah keinginan untuk mengetahui keragaman suatu kelompok data. Salah satu cara untuk mengetahui keragaman dari suatu kelompok data adalah dengan mengurangi setiap nilai data dengan rata-rata kelompok data tersebut, selanjutnya semua hasilnya dijumlahkan.

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["P"].describe()
df["Q"].describe()
df["R"].describe()
df["S"].describe()
print("varian P :",df['P'].count())
print("varian Q :",df['Q'].count())
print("varian R :",df['R'].count())
print("varian S :",df['S'].count())

    Output :

    varian P : 500
varian Q : 500
varian R : 500
varian S : 500

    6. Skewness

    Skewness merupakan derajat ketidak simetrian (keasimetrian), atau dapat juga disefinisikan sebagai penyimpangan dari kesimetrian dari suatu distribusi.

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["P"].describe()
df["Q"].describe()
df["R"].describe()
df["S"].describe()
print("skewness P :",df['P'].skew())
print("skewness Q :",df['Q'].skew())
print("skewness R :",df['R'].skew())
print("skewness S :",df['S'].skew())

    Output :

    skewness P : 0.3023442392819847
skewness Q : 0.11858689559592421
skewness R : -0.12085362312273451
skewness S : -0.2887811407183489

    7. Nilai Minimal

    **9. Quartile **

    Kuartil atau dalam bahasa Inggris disebut dengan Quartile adalah nilai yang membagi sekumpulan data yang terurut menjadi empat bagian yang sama yaitu bagian pertama, bagian kedua, bagian ketiga dan bagian keempat. Terdapat tiga buah Kuartil yang didapati dari suatu gugus data yaitu Kuartil 1 (Q1), Kuartil 2 (Q2) atau Median dan Kuartil 3 (Q3).

    Dari gambar diatas, jelas terlihat bahwa ada empat bagian yang sama dalam sekumpulan data yang terbagi menurut pembagian Kuartil, yaitu :

    • 25% pertama adalah bagian terendah.
    • Bagian 25% berikutnya adalah bagian terendah kedua hingga ke Median.
    • Bagian 25% setelah Median adalah bagian tertinggi kedua.
    • 25% keempat adalah bagian yang tertinggi.

    a. Kuartil 1 (Q1)

    Code :

    import pandas as dt
df=dt.read_csv("data.csv")
df["P"].describe()
df["Q"].describe()
df["R"].describe()
df["S"].describe()
print("Q1 P :",df['P'].quantile(0.25))
print("Q1 Q :",df['Q'].quantile(0.25))
print("Q1 R :",df['R'].quantile(0.25))
print("Q1 S :",df['S'].quantile(0.25))

    Output :

    Q1 P : 2.0
Q1 Q : 3.0
Q1 R : 4.0
Q1 S : 4.0
